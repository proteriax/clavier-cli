# clavier

Change or view input source from the command line.

## Example
```sh
clavier "U.S."
clavier "Hiragana"
clavier "Pinyin - Simplified"

# Show current input source
clavier # U.S.
```

## Build
This process should take only a few seconds.

```sh
git clone https://github.com/proteriax/clavier-cli.git
cd clavier-cli
make all
```
